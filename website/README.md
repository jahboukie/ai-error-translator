## Responsive Design

The landing page is fully responsive with breakpoints at:

- **Desktop**: 1200px+ (full layout)
- **Tablet**: 768px - 1199px (adapted grid)
- **Mobile**: Under 768px (stacked layout)
- **Small Mobile**: Under 480px (optimized for small screens)
